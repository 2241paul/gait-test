# 基于智能手机传感器的十秒踵趾步态平衡测试：原理与临床应用

## 背景

退行性颈脊髓病（Degenerative Cervical Myelopathy, DCM），包括颈椎病、后纵韧带骨化症（OPLL），是成人脊髓受压最常见的病因。脊髓压迫累及后索时，会造成本体感觉障碍，进而导致平衡功能受损。传统临床检查方法多为定性，缺乏便捷的量化评估手段。智能手机内置加速度计和陀螺仪，为门诊便捷量化评估平衡功能提供了可能。

## 哪些神经系统疾病会影响行走平衡？

平衡维持需要**本体感觉、前庭系统、视觉系统**三者的协同整合，任一系统受损均可导致平衡障碍。临床上常见疾病包括：

| 疾病 | 平衡受累机制 | 平衡是否受累 |
|------|----------|------|
| **脊髓型颈椎病/OPLL** | 脊髓后索受压 → 本体感觉障碍 | ✅ 肯定受累 |
| 腰椎管狭窄症 | 机械性压迫神经根，不累及脊髓本体感觉 | ❌ 通常不受累 |
| 膝关节骨性关节炎 | 疼痛导致步态改变，不累及平衡整合 | ❌ 通常不受累 |
| 帕金森病 | 基底节功能障碍 → 姿势不稳 | ✅ 肯定受累 |
| 脑卒中（小脑/脑干） | 直接损伤平衡中枢/前庭通路 | ✅ 肯定受累 |
| 多发性硬化 | 脊髓/脑干脱髓鞘病变 | ✅ 常受累 |
| 周围神经病 | 下肢本体感觉传入障碍 | ✅ 常受累 |

> **临床意义**：**利用平衡测试可以辅助鉴别颈椎病/OPLL与腰椎管狭窄、膝关节疾病**，后二者平衡指标通常正常，而颈椎病常有异常。

## 颈椎病是否会导致行走平衡异常？

**是的，多项研究已经证实：**

- 颈椎病脊髓受压 → **后索（本体感觉通路）受损** → 下肢位置觉、振动觉减退 → 平衡控制能力下降
- 定量研究证实：与健康对照组相比，颈椎病患者静态站立时**总摇摆面积可比正常人增大700倍**，平均摇摆速度也显著增大，平衡异常客观存在
- 即使是轻型颈椎病，也能检测出客观的平衡异常

## 踵趾步态（串联步态，tandem gait）会放大平衡异常吗？

**是的，这已经得到临床研究验证：**

- 原理：**要求足跟紧贴足尖行走**，步幅极小，直线行走，对平衡控制的要求显著提高
- 临床价值：**轻度平衡异常在普通行走中可能无法发现，在踵趾步态下会被放大**，更容易检测出来
- 已有研究：10步踵趾步态试验对退行性颈脊髓病有诊断价值，可以发现已经存在但不易发现的平衡障碍

我们开发的"**十秒踵趾步态平衡测试**"正是基于这一成熟原理，要求患者完成十秒踵趾行走，通过手机传感器检测平衡控制能力。

## 常规临床评估方法有哪些？

| 方法 | 性质 | 说明 |
|------|------|------|
| Romberg 征 | 定性 | 双脚并拢站立，观察睁眼闭眼是否摇晃，主观判断 |
| 10步踵趾步态试验 | 半定量 | 记录能否完成10步踵趾行走，能完成提示平衡功能较好，已是临床推荐检查方法 |
| 神经系统专科查体 | 定性 | 检查 Hoffmann 征、腱反射、感觉减退 |
| mJOA 评分 | 半定量 | 改良日本骨科协会评分，包含下肢功能分项 |
| 测力台（Force Plate） | 定量金标准 | 可以精确测量压力中心（COP）摇摆面积、平均速度，价格昂贵，不适合门诊常规使用 |
| 多传感器可穿戴设备 | 定量研究用 | 腰部和下肢放置多个IMU传感器，准确性高，但操作复杂 |
| **智能手机内置传感器（本文方案）** | **定量便捷筛查** | **利用手机现成传感器，操作简便，适合门诊常规筛查和术后随访** |

## 智能手机传感器评估平衡可行性已有研究吗？

**是的，已有充分研究验证可行性：**

- 智能手机内置**加速度计 + 陀螺仪**，本身就是现成的惯性测量单元（IMU）
- 多项方法学研究验证：**智能手机传感器测量步态/平衡参数与专业设备一致性良好**，组内相关系数ICC > 0.75，满足临床需要
- 在骨科/脊柱外科领域，已有多篇研究探讨了可穿戴/手机传感器在颈椎病术前评估、术后随访中的应用，证实其临床价值

## 本项目方案设计

### 测试流程

1. **姿势要求**：测试者手握手机，**尽可能保持手机稳定**，**足跟紧贴足尖**，沿直线缓慢行走十秒
2. **核心原理**：平衡受损者需要不断调整姿势才能维持重心稳定，会导致握持的手机产生额外晃动
3. **参数提取**：手机传感器采集加速度和角速度信号，计算以下参数：
   - 基本步态：总步数、步频、平均步周期
   - 步态规律性：步态周期变异系数
   - 侧向平衡：侧向摆动标准差反映左右方向平衡稳定性
   - 整体加速度：加速度标准差反映整体加速度波动程度
   - 整体角速度：陀螺仪总体RMS反映整体晃动幅度，平衡越差，晃动越大
   - **综合平衡评分**（0-100）：评分越高，表示平衡稳定性越好

### 临床意义

- **辅助评估疾病严重程度**：综合评分越低，提示平衡受损越严重，疾病越重
- **辅助鉴别诊断**：
  - 颈椎病/OPLL → 平衡受损 → 晃动参数增大 → 综合评分降低
  - 腰椎管狭窄症/膝关节骨性关节炎 → 平衡功能通常不受累 → 晃动参数正常 → 综合评分正常

## 结论

基于智能手机传感器的十秒踵趾步态平衡测试，是一种便捷、无创、免费的平衡功能量化评估方法，可以辅助颈椎病/OPLL的临床评估，帮助评估疾病严重程度，并辅助鉴别诊断表现为下肢行走困难的腰椎管狭窄症、膝关节疾病，适合临床门诊常规使用和患者术后随访。
