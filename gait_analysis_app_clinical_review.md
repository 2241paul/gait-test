## 2. 参数提取与DCM步态异常特征匹配性评估

### 当前提取参数与文献对比

根据文献综述，DCM步态异常的核心参数特征为：
- **步速减慢**
- **步态变异性增加**
- **步频降低**
- **稳定性下降**

开发者实际提取的参数：

| 参数名称 | 是否提取 | 文献是否需要 | 匹配DCM病理生理 |
|----------|----------|--------------|-----------------|
| 10秒总步数 | ✅ | 必需 | 直接反映运动能力，步数越少损伤越重 |
| 步频（步/分钟） | ✅ | 必需 | DCM患者步频显著降低，与mJOA相关 |
| 步态周期变异系数 | ✅ | 必需 | DCM患者变异增大，反映脊髓调控受损 |
| 振幅变异（对称性） | ✅ | 推荐 | DCM患者步态对称性下降 |
| 综合稳定性评分 | ✅ | 新增 | 综合反映步态稳定性，临床直观 |
| 平均步态周期 | ✅ | 推荐 | 用于计算变异系数 |
| 估计步速 | ✅ | 必需 | DCM患者步速减慢 |
| **平均步长** | ❌ | 必需 | 当前仅估算，无法直接测量 |
| **站立相/摆动相比例** | ❌ | 推荐 | DCM站立相比例增加 |
| **初始峰值加速度** | ❌ | 推荐 | 文献指出此参数比单纯摇摆更敏感 |
| **躯干摇摆幅度** | ❌ | 推荐 | DCM患者摇摆幅度增大 |

### 匹配性评价

**已满足的核心要求**：
- ✅ 成功提取了**步频**、**步数**、**步态周期变异系数**三个核心预测因子，这些参数与DCM下肢功能受损特点完全匹配
- ✅ **步态变异性**的提取非常关键，这是DCM脊髓调控受损的直接客观表现，当前实现正确
- ✅ 综合稳定性评分的构建符合临床认知，稳定性下降是DCM核心特征

**存在的不足**：
- ⚠️ 缺少**平均步长**的直接测量（当前是基于步频估算）。虽然十秒直线步行难以通过手机传感器直接计算步长，但文献明确指出步长缩短是DCM重要特征
- ⚠️ 未利用陀螺仪数据提取躯干摇摆幅度等动力学参数，这是DCM平衡受损的直接表现

### 结论

**整体评价**：✅ **基本满足需求，但核心参数不完整**。已提取的参数均与DCM下肢功能受损特点高度匹配，但缺少几个文献推荐的重要参数。

## 3. mJOA下肢功能评分预测评估

### 预测模型分析

当前实现：
- 输出预测范围：**0-4分**，与标准mJOA下肢评分范围一致 ✅
- 预测因子：步频 + 稳定性评分 + 总步数，三者均为文献证实与mJOA显著相关 ✅
- 基于临床知识的规则模型，逻辑清晰可解释 ✅
- 预测逻辑符合文献报道：
  - 步频越低 → 评分越低（功能越差）
  - 稳定性越差 → 评分越低
  - 步数越少 → 评分越低

**评分截断值分析**：

| 截断规则 | 临床合理性 | 文献支持 |
|----------|------------|----------|
| 步频 < 80 → 减1分 | 合理 | 正常人约100-120步/分钟 |
| 步频 < 60 → 再减1分 | 合理 | 严重受损者步频显著降低 |
| 稳定性 < 50 → 减1分 | 合理 | DCM稳定性显著下降 |
| 稳定性 < 30 → 再减1分 | 合理 | |
| 步数 < 8（10秒）→ 减1分 | 合理 | 正常人约15-20步 |

### 有效性评价

**优势**：
1. 模型基于文献证实的预测因子，临床逻辑正确
2. 规则简单透明，易于理解和验证
3. 输出与标准mJOA下肢评分范围完全一致（0-4分）

**局限性**：
1. 当前是**基于临床知识的规则模型**，尚未利用已有的1541例多中心临床数据进行训练
2. 缺少模型验证（MAE、RMSE、准确率等指标）
3. 未纳入步长、步态变异系数等重要预测因子

### 结论

✅ **预测逻辑正确，基本满足临床需求**。但目前是基于经验规则的预测，建议进一步利用现有临床数据训练机器学习模型提高预测精度。

### 具体改进建议

#### A. 参数提取方面（优先级：中）

1. **补充平均步长估计**：虽然手机无法直接GPS测量，但可基于步频和身高进行个体化估算：
   - 平均步长 ≈ 身高 × 0.45（经验系数）
   - 这样可以获得步速 = 步频 × 步长 / 60，比当前估算更准确

2. **利用陀螺仪数据**：当前代码采集了陀螺仪但未使用，建议提取：
   - 躯干前后/内外摇摆幅度标准差
   - 反映DCM平衡功能受损，提高预测精度

3. **添加站立相比例估计**：可基于加速度峰值检测脚跟着地/脚尖离地事件，计算站立相占比，这是DCM重要异常特征

#### B. 预测模型方面（优先级：高）

1. **利用现有1541例多中心数据训练模型**：
   - 当前是经验规则模型，建议使用现有数据训练线性回归/随机森林模型
   - 可大幅提高预测准确性
   - 需要报告训练集/验证集MAE、RMSE、R²等指标

2. **纳入更多预测因子**：将步态周期变异系数也纳入预测模型，文献证实其与mJOA独立相关

#### C. 功能完善方面（优先级：高）

1. **添加第二种测试模式**：
   - 当前仅支持十秒直线步行，建议增加**十秒原地踏步测试**选项
   - 文献证实十秒踏步测试在DCM中应用更广泛，循证医学证据更充分
   - 适合门诊空间狭小或重度功能障碍患者
   - 两种模式可以切换，提高临床适用性

2. **添加历史结果对比功能**：
   - 使用localStorage保存历史测试结果
   - 支持随访过程中对比变化，量化评估术后恢复情况

3. **结果报告优化**：
   - 当前只显示预测评分，建议增加各项参数的临床解读
   - 例如："步态变异系数18.5%，提示步态稳定性下降，符合颈脊髓受压表现"

4. **手机放置位置提示**：
   - 当前提示"放在口袋或握在手中"，建议更明确：推荐放在腰部裤子口袋中（靠近重心）

#### D. 临床验证方面（优先级：中）

1. 建议开展小样本临床验证研究：
   - 纳入一定数量DCM患者，同时进行小程序测试和医生mJOA评分
   - 计算一致率、组内相关系数(ICC)等验证指标
   - 为临床应用提供更充分的证据
